# 재난 대피 경로 안내 시스템

가천대학교 AI공학관 3~5층에서 화재 발생 시, 사용자의 현재 위치에서 가장 가까운 비상구까지 최단 탈출 경로를 실시간으로 안내하는 시뮬레이터입니다.
화재·연기가 매 시간 단위로 확산되며, 경로가 차단되면 자동으로 재탐색합니다.
5층에서 출구가 모두 막힐 경우 계단을 통해 아래 층으로 내려가는 층간 이동도 지원합니다.

## Streamlit UI 주요 기능

발표 시연 흐름은 **설정 → 재생** 두 단계로 나뉩니다.

### 1) 설정 단계

- 층 선택 (3 · 4 · 5층)
- 지도 클릭으로 **내 위치(S)** 지정
- **화재 발생 방식 2가지**
  - *랜덤*: 화재 지점 수(1~5)만 정하면 무작위 발화
  - *직접 지정*: 화재 위치를 지도 클릭으로 직접 지정 (다시 클릭 시 해제)

### 2) 재생 단계

- 시간 `t`가 **자동으로 0 → 1 → 2 …** 진행 (재생/일시정지/다음 단계/속도 조절)
- 화재·연기가 매 시각 함께 확산 (화재 ~0.4 m/s, 연기 ~5 m/s)
- **대피자가 매 스텝 현재 화재 기준으로 경로를 다시 잡으며 한 칸씩 이동**
- **연기 지연**: 연기 구역 통과 시 1턴 추가 소요
- **층간 이동**: 5층 출구가 모두 차단되면 계단(T)을 통해 4층 → 3층으로 자동 하강
- **탈출 실패 애니메이션**: 안전 경로 소실 시 불을 피해 이동하다 포위되는 과정 표시
- **탈출 성공 시**: 화재에 가려졌던 지나온 경로까지 지도에 표시

### 3) 우측 정보 패널

- 현재 층 · 경과 시간 · 출구까지 남은 거리
- 현재 경로 위험도 점수 및 등급 (안전 / 주의 / 위험)
- 전체 대피 통계: 랜덤 10명 배치 후 퀵 정렬 + 이진 탐색으로 목표 시간 내 탈출 인원 집계
- 출구 혼잡도 분석: 출구 처리 용량 설정 후 FIFO 큐 기반 대기 시뮬레이션
- **위험도 히트맵**: 화재 위험(65%) + 경로 병목(35%) 결합 색상 오버레이

## 동작 흐름 (알고리즘)

```
1. 사용자가 맵에서 현재 위치 클릭 → S 설정 / 화재 지점 지정(랜덤·직접)
2. BFS로 화재 확산 시뮬레이션 (fire_speed=0.13: 사람보다 7.5배 느림)
3. BFS로 연기 확산 시뮬레이션 (smoke_speed=1.7: 사람보다 1.7배 빠름)
4. 다중 층 A*로 전체 대피 경로 미리보기 계산 (5→4→3층 포함)
5. 유니온-파인드로 S~X 연결 여부 실시간 확인
6. A*로 최단 탈출 경로 탐색 (도착 시점 기준 화재 차단)
7. 매 시각 t마다 현재 위치·화재 기준으로 경로 재탐색하며 한 칸 이동
8. 계단 도달 시 아래 층으로 전환, 아래 층 기준으로 재탐색
9. 안전 경로 소실 시 best-effort 이동 → 포위되면 탈출 실패
```

## 핵심 알고리즘 설명

### A* — 도착 시점 기준 화재 차단

단순히 현재 시각 기준이 아닌, 해당 셀에 실제로 도착하는 시간(`current_time + 걸음 수`)과 화재 도달 시간을 비교합니다.
이동 중 화재에 따라잡히는 경우를 정확히 차단합니다.
`fire_time == -1`은 화재 미도달(안전)로 처리합니다.

```python
# 도착 시점 기준 — -1(미도달)은 안전, 양수면 따라잡힘 여부 비교
if fire_time[nr][nc] != -1 and fire_time[nr][nc] <= current_time + new_g:
    continue
```

### 다중 층 A* — 층간 이동

노드를 `(층, 행, 열)` 3차원으로 확장하여 계단 셀에서 아래 층으로 전이하는 에지를 추가합니다.
5층 출구가 모두 화재로 차단될 때 자동으로 계단 경유 경로를 탐색합니다.

```python
# 계단 셀 도달 시 아래 층으로 전이 (하강만 허용)
if (r, c) in stair_set:
    lower_floor = max(f for f in floor_grids if f < floor_num)
    nnode = (lower_floor, r, c)
```

### BFS 화재·연기 확산

여러 발화 지점을 BFS 큐에 동시 삽입해 자연스러운 동시 확산을 구현합니다.
`fire_speed` / `smoke_speed` 파라미터로 확산 속도를 조절합니다.

```python
fire_time, _ = simulate_fire_spread(grid, positions, fire_speed=0.13)
smoke_time   = simulate_smoke_spread(grid, positions, smoke_speed=1.7)
```

### 퀵 정렬 + 이진 탐색 — 탈출 통계

랜덤 10명의 탈출 시간을 퀵 정렬로 정렬한 뒤, 이진 탐색으로 목표 시간 이하 탈출 인원을 O(log n)에 집계합니다.

```python
sorted_times = quick_sort_escape_times(escape_times)   # 퀵 정렬
count = binary_search_time(sorted_times, target_time)  # 이진 탐색
```
